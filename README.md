# IUT Web Course Lab Homeworks

## Homework 1
#### A simple node.js server
Simply open a linux terminal:
```bash
git clone https://github.com/alirezaomidi/web-course-lab
cd web-course-lab/hw1
node server.js
```
Then use a browser to open the `index.html`.

## Homework 2
